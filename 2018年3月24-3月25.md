> 控制参数数量

超过3个参数，建议使用对象传参。不必记忆参数传入的顺序，易于扩展
```
encryptStr: function (obj) {
        var _default={
            type:0,
            replacement:'*'
        };
        for(var key in obj){
            _default[key]=obj[key];
        }
},

//调用方式
ecDo.encryptStr({str:'18819266335',regArr:[5],type:0,replacement:'-'});

```
